# Student Registration Status Checker

## 📌 Purpose
This project is a simple program that checks a student's registration status based on the number of units they have registered.

## ⚙️ How the Program Works
- The user enters the student name
- The user enters the number of registered units
- If the units are greater than 7, the system displays:
  "Overload - Approval Required"
- Otherwise, it displays:
  "Registration Accepted"
- Finally, a summary is shown with the student name, units, and status
